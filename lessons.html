what I have learnt

Ghana Tech Lab Web Development Training @ Ho Node Hub

We started the course by setting up our development environment in VS Code, set up and connected our GitHub accounts to VS Code.
I learnt basic concepts about HTML and CSS. HTML as a markup language helps us to put text or information on websites.
CSS helps us to style the webpage and determine how items look on the page.

We set up our first GitHub repository and initialized it.
We set up our GitHub site and used the Gitbash terminal to push our code to the site using the add, commit and push commands.

We practised with HTML today.
We learnt how to set up our boiler plate, worked with tags and began to use what we learnt to develop a simple html website.

I learnt about website design using Adobe XD today.
For any web development project, the first step is to picture what you want to design and bring it to live using design software like Adobe XD and Figma. This will help you as you begin to actually design your website in yur editor.

We delved today into CSS. CSS is a powerful tool for making websites look beautiful.
It contains numerous properties that control how html elements are displayed.
Some important properties we learnt about were width, height, padding, vh, margin and flex.

We continued building our portfolio website today!
We worked on our navigation bar, which holds all the links to our pages.
I experienced the effect of properties like flex, display, margin, and padding and how they affect elements on a webpage.

We learnt about HTML forms today. We used the form tag to create a simple form with two inputs and a textarea.
We also worked on the lessons page of our portfolio website, which involved the use of javascript to manipulate a button to drop down and reveal text beneath it.

Today's project was about creating a To-do List using HTML, CSS and javascript.
The list was created usng the input and button tags, and javascript was used to manipulate the "add" button to add a new todo item and also delete items. The ability to mark tasks as completed was also added.

We started an interesting project today that relied heavily on javascript. It was a project of Flash Cards.
We created buttons, inputs and a transparent button.

We continued the Flash Cards project, styling our page and adding javascript code to control the fnctioning of the flash card.
We also learnt how to import use fontawesome icons and fonts in our projects.

We were introduced to Bootstrap today. We started an image slider project using bootstrap.

We were tasked the previous day to finish styling the image slider project we started on our own. We each presented our results in the class, which had to be fully responsive.

We started another project to design a website with sections and cards. This was also a bootstrap project. It was very interesting and involving. We were again tasked to make the project responsive on our own as a homework.

We worked on updating information on our portfolio websites. We connected all our projects and also did debugging among other fixes.
